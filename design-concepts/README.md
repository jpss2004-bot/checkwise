# `design-concepts/` — Active design inspiration

In-flight design references for the CheckWise redesign work. Treat this
as *inspiration*, not as a source of truth.

## Contents

- `inspo-screenshots/` — curated UI screenshots from external products
  (Linear, Stripe, Mercury, Ramp, and similar premium SaaS) plus
  redesign mood frames. Captured during the V1.4–V1.6 polish passes.

## How agents should use this

- Read these images to understand the *visual register* CheckWise aims
  for: precise, calm, dense, premium legal-tech.
- Do **not** translate any single screenshot 1:1 into product UI —
  CheckWise has its own tokens (`frontend/app/globals.css`) and product
  surfaces (see `docs/DESIGN_SYSTEM.md`).
- Banned: pasting these as static HTML, using them as visual decoration
  in product surfaces, or treating their layouts as binding.

## What this is *not*

- **Not** a logo folder. Logos live at `../brand_assets/Logos CW/`.
- **Not** a token store. Tokens live at `../frontend/app/globals.css`.
- **Not** post-redesign archive. When inspiration stops being live
  reference, move it to `_reference/inspo-archive/` at the workspace root.

See also: `../DESIGN.md`, `../docs/DESIGN_SYSTEM.md`,
