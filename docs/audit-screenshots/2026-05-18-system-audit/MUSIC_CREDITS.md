Track source: <https://incompetech.com/music/royalty-free/index.html?isrc=USUAN1100765>

The track is mixed at approximately -22 dB with sidechain ducking
under the Spanish voice-over, so the original recording is heard
quieter than in the source file.

## Replacing the track

To use a different track, replace `music.mp3` in this directory
with any audio file ffmpeg can decode (MP3, WAV, M4A, FLAC) and
re-run `scripts/finalize_demo.py`. If you use a third-party track,
update this file with the correct attribution before publishing
the demo externally.
